src/@terminus/ngx-tools/src/debounce/debounce.ts

debounce
debounce(func: typeReference, wait: number)

Returns a function, that, as long as it continues to be invoked, will not be triggered. The function will be called after it stops being called for N milliseconds.

Parameters :
Name Type Optional Description
func typeReference
  • The function to call after the debounce period
wait number
  • The length of time to wait between calls (ms)

Name Type Optional Description

const arr = [{a: &#39;foo&#39;, b: 1}, {a: &#39;bar&#39;, b: 6}, {a: &#39;foo&#39;, b: 6}];
